# Library Services Overview

Nexus library services provide core agent logic, channel routing, and automation.

## Features

- Agent reasoning and action execution
- Channel integration via OpenClaw
- Workflow automation and scheduling

## Structure

- `/src/services`: Core service logic
- `/src/agents`: Agent orchestration
- `/src/channels`: Channel plugins and routing
